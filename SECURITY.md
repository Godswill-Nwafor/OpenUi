# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.x     | Yes       |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please email us at **security@openui-hub.dev** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations

We will acknowledge your report within **48 hours** and provide a detailed response within **7 days**.

## Disclosure Policy

Once a fix is deployed, we will:

1. Credit the reporter (unless you prefer to remain anonymous)
2. Publish a security advisory on GitHub
3. Update the CHANGELOG with the fix details

Thank you for helping keep OpenUI Hub secure.
