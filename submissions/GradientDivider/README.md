# GradientDivider

A slim animated gradient line for visually separating page sections — a decorative alternative to a plain `<hr>`.

## Purpose

`GradientDivider` gives you a colorful, subtly animated horizontal rule to break up long pages (between hero and features, above a footer, etc.) without needing any JavaScript or a framework.

## Installation

```bash
# No installation needed — copy GradientDivider.html and GradientDivider.css into your project
```

## Usage

```html
<link rel="stylesheet" href="GradientDivider.css" />

<div class="gradient-divider" role="separator" aria-orientation="horizontal"></div>
```

## Dependencies

- None (pure HTML + CSS, no JavaScript)

## Example

```html
<section>...</section>

<div class="gradient-divider" role="separator" aria-orientation="horizontal"></div>

<section>...</section>
```
